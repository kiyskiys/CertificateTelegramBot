# CertificateTelegramBot

Telegram-бот для автоматизации заказа справок с административной панелью.
---
Основной функционал:

***Пользовательские функции***

- регистрация (ФИО, группа, телефон);

- заказ справок (стандартные, специальные);

- просмотр статуса заявок;

- получение уведомлений о готовности.


***Административные функции***

- управление заявками (просмотр, смена статуса);

- управление доступом пользователей (авторизация, назначение ролей).

---

Стек технологий

Язык	- C#, .NET 8

Telegram API - Telegram.Bot

ORM	- Entity Framework Core

База данных -	SQLite

---

Создать файл конфигурации appsettings.json в директории проекта.

```
{
  "BotConfiguration": {
    "BotToken": "YOUR_TELEGRAM_BOT_TOKEN",
    "PreAdminIds": [
      1234567
      ]
  }
}
```
И запутсить проект, файл БД будет создан автоматически при первом запуске.
